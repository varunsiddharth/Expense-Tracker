# Expense Tracker

A modern expense tracker web application built using HTML, CSS, and JavaScript.

## Features

- Add expenses with category, date, and notes
- Dynamic dashboard cards
- Total expense tracking
- Total entries counter
- Top category display
- Delete expenses
- Data stored using localStorage
- Responsive and modern UI
- Live deployed website

## Technologies Used

- HTML
- CSS
- JavaScript
- localStorage

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

## Challenges Faced

- Managing localStorage data
- Updating dashboard cards dynamically
- Designing a modern UI layout
- Handling delete functionality properly
